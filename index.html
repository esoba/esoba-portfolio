Evaluating the Stability of Non-Adaptive Trading in Continuous Double Auctions

I am working with the Strategic Reasoning Group (SRG) through the University of Michigan CSE department on a replication paper that explores Reinforcement Learning agents in a simulated stock market. My responsibilities include rewriting existing code in tensorflow, interacting with a Java based market simulator, and training a Deep Q-Learning agent to pick optimal parameters for trading. Since starting the project, I have: designed a Zero Intelligence agent that trades using a dummy policy based on private evaluations of securities, created functions to bin and parse data from simulated market, and designed a Tabular Q agent that continously learns and trades in the market using an epsilon greedy policy. All of the work so far has been done in tensorflow with python and Java. My next steps are to perform a a parameter sweep on the University of Michigan's supercomputer cluster and begin designing the DQN agent.

Dorsch Heart Failure

I worked with the Dorsch Heart Failure Project performing time series statistical analysis on 43 patients who currently suffer from symptoms of heart failure. The purpose of the project was to see whether daily self monitoring of symptoms can lead to proper medical intervention before symptoms worsen. I created a model to predict a given patient's well being based on previous day(s) data and visualizations of patient statistics. This was my first exposure to working with messy data sets and performing a lot of data engineering. I trained a suite of binary classification models (including Random Forests, SVMs, and decision trees) and used cross validation to determine that a SVM with RBF kernel was the best classifier for this dataset (with ROC AUC metric). Before my time with the project ended, I was able to achieve a 91% accuracy on the training data and 75% accuracy on the testing data. All analyses were done using scikit-learn, matplotlib, and pandas.

Smart Recognition of Neurosurgical Tools

Through the University of Michigan College of Engineering Multidisciplinary Design Program, myself and a team of 5 other Michigan students worked for Stryker to design a system that automatically recognized neurosurgical tool attachment combinations. The goal of the project was to decrease the workflow during surgery that occurs because of the need to calibrate parameters of different tools. I worked on the Computer Vision side of the project, where we used Python, TensorFlow, Keras, OpenCV, and Amazon EC2 private GPU servers to create an image recognition system for burs/attachments (the components of the tool). We achieved a maximum of 95% validation accuracy on a dataset that is comprised of both original pictures as well as augmented ones. We approached this multi-class computer vision problem by using convolutional neural networks built on top of the ImageNet 2014 winning VGG16 architecture.
